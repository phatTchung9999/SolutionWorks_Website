# SolutionWorks

SolutionWorks is a non-profit web project that I am building to help small local businesses run their own websites for free.

The goal of SolutionWorks is to support local communities by providing simple, ready-to-use website templates for small businesses such as barbershops, nail salons, and restaurants. These templates will include practical features like appointment booking, basic business information, and service listings.

Through SolutionWorks, local business owners can choose a template that fits their needs, and I help them set up a professional website without cost or technical barriers. This project is focused on accessibility, simplicity, and real-world usefulness rather than profit.
